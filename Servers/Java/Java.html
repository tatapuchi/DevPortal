Java

A language.

Cross Platform

Write once, Run anywhere. Java's platform independent code doesn't need to be tweaked to run on a different OS.

Multithreaded

Java's multithreading support allows for numerous tasks can be executed simultaneously.

Memory Management

Memory is managed by the garbage collector, meaning that you wont have to stress on destructing objects.

Object Oriented

The object oriented paradigm allows for modular programs with reuseable code

Performance

Although having numerous improvements over time, still inferior to natively compiled lanuages like C or C++

Verbosity

No surprise that Java is verbose, leading to problems with understanding code in larger programs. Although its no C++, Java isn't the most concise language either, with numerous alternatives in the market.

Java

Java is a robust and secure general purpose programming language thats been around since 1995. It is used extensively in enterprise and server applications, and has dominated the Android development sector until the release of Kotlin, and is still a respectable language with quite the grip on the industry.




Kotlin

Interested in android development? Java just isnt cutting it anymore? Try out Kotlin, a concise, nullsafe alternative to Java that has taken the Android development sector by storm. Google's preffered language for Android, and just a small jump from Java.
